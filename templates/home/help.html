Welcome to the Task Manager for IT Companies

An intuitive way to manage tasks, projects, and teams efficiently.

Navigation

In the navigation menu, you will find the following sections:

  • Home
  • Task Manager
  • Project Manager
  • Team Manager

Home Page

Your personal space where you can see your task statistics, both completed and in progress.

Additionally, there is a dedicated table displaying tasks assigned specifically to you.

At the bottom of the page, you will find a table with information about other employees. Clicking on their username allows you to view their profile.

My Profile

Here, you can edit your profile and update your personal information.

Task Manager

You can flexibly create tasks, assign different executors, teams, and link tasks to specific projects.

Team Manager

Create teams with an unlimited number of participants and assign projects for them to work on collaboratively.

Project Manager

Easily create projects with descriptions and access detailed project information.

Full Control

In all sections mentioned above, you have full control to Create, Edit, Delete, and Manage as you wish.
